# pdf2cbx

Simple shell script to convert PDF to CBR/CBZ archive.

INSTALL :

- Simply move this script in /usr/local/bin and use "chmod +x"

REQUIRE :

-  pdftocairo, rar and zip commands, use pdftocbx -t to test requirements
-  pdftocbx -h print help
